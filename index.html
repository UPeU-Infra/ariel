El Problema

Diez plataformas,
cero conexion

Cada universidad opera multiples sistemas desconectados: repositorio de tesis, revistas academicas, catalogo de biblioteca, sistema academico, LMS, ERP financiero, directorio de correo...

Los estudiantes no saben donde buscar. Los docentes pierden tiempo navegando plataformas. El helpdesk responde las mismas preguntas una y otra vez.

El resultado: frustacion, informacion perdida, plataformas subutilizadas y una experiencia fragmentada que no esta a la altura de lo que los usuarios esperan en 2026.

Las soluciones existentes (EBSCO EDS, Ex Libris Primo) cuestan $20,000-80,000/ano, solo cubren busqueda bibliografica, y no ofrecen chat conversacional ni acceso a datos institucionales.

La Solucion

GUIA conecta
todo

🔍

Busqueda semantica con RAG

No keywords — lenguaje natural. GUIA entiende "que tesis hay sobre machine learning en educacion" y busca en todos los repositorios conectados.

💬

Chat multicanal

Web, Telegram, WhatsApp, Teams. Los estudiantes preguntan donde ya estan — no necesitan aprender otra plataforma.

🔌

Conectores modulares

Cada sistema es un modulo independiente. Instala solo los que necesitas. Crea los tuyos para sistemas custom.

🤖

MCP Server + API REST

Expone tu corpus institucional a agentes AI externos (Claude, GPT, Perplexity). Tu produccion cientifica visible en el ecosistema AI.

🏛️

Hub federado (para redes)

Consorcios y redes pueden federar multiples Nodes para busqueda unificada de investigacion. Compatible con OpenAIRE, LA Referencia, ALICIA.
