El Problema

Una escala sin
infraestructura

El sistema educativo adventista constituye el segundo mayor sistema educativo privado del mundo con 118+ universidades en más de 50 países.1 Sin embargo, menos del 5% cuenta con un repositorio institucional operativo y no existe ninguna red federada entre ellas.2

Entre 5,000 y 15,000 publicaciones académicas anuales son actualmente invisibles para Scopus, WoS, OpenAlex y para otros investigadores adventistas en cualquier parte del mundo.2

El repositorio de Andrews University recibió visitas de 191 países y superó el millón de descargas en sus primeros tres años de operación.3 Esto sugiere que actúa como repositorio de facto para toda la denominación — dado que la gran mayoría de universidades adventistas no tiene infraestructura propia, Andrews concentra la visibilidad de la producción académica global adventista en una sola institución.

⚠ Riesgo sistémico

Si Andrews University cambia su política o plataforma, la visibilidad de toda la producción científica adventista global colapsa. La producción de África, Asia y Latinoamérica no tiene plataforma propia.4

AWS · EC2 t3.xlarge · S3 · PostgreSQL

Desarrollado por CERN, usado por Zenodo y TU Graz.7 Docker Compose — stack conocido. UI moderna. OAI-PMH nativo incorporado. Open source MIT. El más rápido de desplegar para el piloto.
